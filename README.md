#### Change Management

  * Define a System State
  * Enforce the system State
  * Example :

    * Apache web installed
    * Apache web at version xxx
    * Apache web started.

#### Provisioning

  * Prepare a system to make it ready
  * transition of system from one state to different state.
  * Example :

    * Make an FTP server
    * Make an Apache server
    * Make an NFS server.
